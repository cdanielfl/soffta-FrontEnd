# Pasta de Arquivos Públicos

## Campainha Hospitalar

Coloque seu arquivo de áudio da campainha aqui com o nome:
**`hospital-bell.mp3`**

### Formatos suportados:
- `.mp3` (recomendado)
- `.wav`
- `.ogg`

### Onde encontrar sons de campainha hospitalar:
1. **Freesound.org** - https://freesound.org/search/?q=hospital+bell
2. **Zapsplat** - https://www.zapsplat.com/
3. **Pixabay** - https://pixabay.com/sound-effects/

### Dica:
Procure por termos como:
- "hospital bell"
- "hospital chime"
- "medical notification"
- "ding dong hospital"

### Se quiser usar outro nome de arquivo:
Edite o arquivo `src/utils/audioUtils.js` e altere a linha:
```javascript
const audio = new Audio('/hospital-bell.mp3');
```
